# Angular 简介
## Angular 是什么？
    Angular 是由谷歌开发与维护一个开发跨平台应用程序的框架，同时适用于手机与桌面
## Angular有什么特点 
    · 基于Angular 我们可以构建适用于所有平台的应用。比如：Web应用、移动web应用、移动应用和桌面应用等。
    · 通过Web Worker和服务端渲染（SSR）,达到在如今Web平台上所能达到的最高渲染速度。
    · Angular 让你能够有效掌握可伸缩性。基于RxJS、Immutable.js和其它推送模型，能适应海量数据需求。
## Angular提供了哪些功能
    · 动态HTML
    · 强大的表单系统（模板驱动和模型驱动）
    · 强大的视图引擎
    · 事件处理
    · 快速的页面渲染
    · 灵活的路由
    · HTTP服务
    · 视图封装
    · AOT、Tree Shaking
